# הכספת המשפחתית 🏦

אפליקציית ניהול תקציב משפחתי.

## הרצה מקומית
```bash
npm install
npm run dev
```

## פריסה ל-Vercel
1. דחוף את הקוד ל-GitHub
2. חבר את ה-repo ב-vercel.com
3. Vercel יזהה אוטומטית את Vite ויבנה

## הגדרות Build ב-Vercel
- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
